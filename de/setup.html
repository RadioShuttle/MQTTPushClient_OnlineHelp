App einrichten

Wenn die App zum ersten Mal gestartet wird, werden Sie gefragt, ob die App Benachrichtigungen senden soll. Sie sollten dies mit Allow bestätigen:

  • Tippen Sie nach dem Starten der App auf das Symbol Edit und dann auf Add new server und geben Sie im Fenster „Add Server“ „New Server“ die Adressen des Push-Servers und des MQTT-Servers („Host“), Ihren Benutzernamen („User“) sowie das zugehörige Passwort („Pwd“) ein. Im Fenster der Serverkonfiguration sind bereits die Adressen von Push- und MQTT-Server vorgegeben. Diese Adressen können mit den von Ihnen gewünschten Serverdaten überschrieben werden.
Hinweis:

Der Standard-Port für den Push-Notification-Server ist 2033. Diese Angabe kann entfallen, wenn der Push-Server unter diesem Port erreichbar ist. Andernfalls muss der Port an den Domain-Namen angehängt werden, z. B. „yourpushserver.de:2009“.

  • Tippen Sie auf Save. Save. Bei unverschlüsselten Verbindungen, oder wenn die Authentizität eines Push-Servers nicht bestätigt werden kann, erscheint eine Warnmeldung. Siehe dazu auch Abschnitt Sicherheit.

Dark Mode / Light Mode

Viele Mobiltelefone unterstützen einen sogenannten Dark- bzw. Light-Modus. Standardmäßig nutzt diese App die Systemeinstellung des Mobiltelefons. Sie können diese Einstellung allerdings auch direkt in der App anpassen. Damit wird die Systemeinstellung des Mobiltelefons überschrieben.

  • Öffnen Sie dazu den Eintrag „Theme“ im Überlaufmenü () und wählen Sie den gewünschten Modus aus (links light, rechts dark).

  • Tippen Sie auf das Icon und wählen Sie den gewünschten Modus aus (links light, rechts dark).

Hinweis:

Mehrere Anwender können ein identisches Konto verwenden. Beispielsweise können alle Mitglieder einer Familie oder Gruppe automatisch die identischen Einstellungen nutzen und gleichzeitig benachrichtigt werden.
