Auf Topics registrieren

Für den Empfang von Nachrichten ist es erforderlich, sich auf mindestens ein Topic zu registrieren.

  • Drücken Sie im Fenster „MQTT Push Client“ so lange auf den gewünschten Servereintrag („Long Press“) bis das Icon in der Menüleiste erscheint und tippen Sie darauf.Tippen Sie im Fenster „MQTT Push Client“ auf Edit und dann auf den Servereintrag.
  • Tippen Sie dann im Fenster „Edit Server“ auf MQTT TOPICS. MQTT Topics.
  • Tippen Sie im Fenster „Topics“ auf Add new topic und geben Sie das Topic an, auf das Sie sich registrieren möchten.
  • Wählen Sie im Abschnitt „Notifications“ aus, wie Sie benachrichtigt werden möchten.

Der Nachrichteninhalt für das angegebene Topic kann gefiltert werden. Sie können eine JavaScript-Funktion hinzufügen, die dann auf jede angezeigte Nachricht angewandt wird.

  • "Tippen Sie unter „Filter Script“ auf ADD bzw. EDIT, um in den JavaScript-Editor zu gelangen. Siehe dazu auch das Kapitel Filterskripte für Topics.
Hinweis:

Die Topics, auf die Sie sich registrieren können, müssen Ihnen bekannt sein. Das bedeutet, die Topics müssen auf Ihrem MQTT-Server angelegt worden sein, und Sie müssen über Lese- und ggf. Schreibberechtigung für diese Topics verfügen.

  • Tippen Sie auf SAVE. Save.

Jetzt haben Sie sich auf ein Topic registriert, über das Sie per Benachrichtigung und Tonsignal (je nach Einstellung) informiert werden.

Hinweis:

Viele MQTT-Server antworten nicht mit einer Fehlermeldung, wenn Sie keine Berechtigung für ein Topic besitzen oder das Topic nicht existiert. Daher kann es sein, dass ein Topic in der App angezeigt wird, Sie aber keine Nachrichten dafür erhalten.
