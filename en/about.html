About MQTT Push Client

“MQTT Push Client” is a remote control app for MQTT environments. It allows displaying MQTT messages and provides a user interface for displaying custom controls (Dashes), which interacts with MQTT messages. In addition, it can issue Apple/Google push messages to issue alarm notifications to iOS and/or Android mobile devices for specified topics.

After a one-time setup on the mobile device, push notifications are automatically received, even if the app is not running. Multiple mobile devices for the same account can be added.

The app interacts with the RadioShuttle MQTT push server which in turn communicates with the user's MQTT server (Broker) to collect and handle MQTT messages for defined topics.
