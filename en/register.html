Register for Topics

For the reception of messages it is necessary to register for at least one topic.

  • In the “MQTT Push Client” window perform a long press on the respective server entry until the icon appears in the menu bar and tap on it.In the “MQTT Push Client” window tap on Edit and then on the respective server entry.
  • In the “Edit Server” window tap on MQTT TOPICS. MQTT Topics.
  • In the “Topics” window tap on Add new topic and specify the topic on that you want to register.
  • In the “Notifications” section choose the way you wish to be notified.

Der Nachrichteninhalt für das angegebene Topic kann gefiltert werden. Sie können eine JavaScript-Funktion hinzufügen, die dann auf jede angezeigte Nachricht angewendet wird. Tippen Sie unter „Filter Script“ auf ADDADD bzw. EDITEDIT, um in den JavaScript-Editor zu gelangen.

Note:

The topics you can register for, need to be known to you. This means that the topics must already exist on your MQTT server, and you must have read and write access for these topics.

  • Tap on SAVE. Save.

Now you have registered for a topic about which you will be informed by notification and sound signal (depending on the setting).

Note:

Many MQTT servers do not response with an error message if you do not have authorization for a topic or the topic does not exist. Therefore it is possible that a topic is displayed in the app, but you do not receive any notification for that.
