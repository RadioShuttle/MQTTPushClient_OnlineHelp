Set Up the App

When the app is started for the very first time, you will be asked if the app is allowed to send notifications. You should confirm this by tapping on the Allow button:

  • After the app has been started, tap on the icon Edit icon and on Add new server and specify the push server address, the MQTT server address (“Host”), your user name (“User”) and the corresponding password (“Pwd”) in the “Add Server” “New Server” window. The addresses of the push and MQTT server are already preset in the server configuration window. These addresses can be overwritten with the desired server data.
Note:

The standard port for the push notification server is 2033. This specification can be omitted if the push server is accessible under this port number. Otherwise the port number must be appended to the domain name, e.g. “yourpushserver.de:2009”.

  • Tap on Save. Save. When using unencrypted connections, or if the authenticity of a push server cannot be confirmed, a warning message is issued. See also Security.
